# 🔎 CCSeeker

A Streamlit-based AI tool that helps you **discover and analyze YouTube creators** for outreach and collaboration.  
It combines the YouTube Data API with Google Gemini to provide:
- Smart channel search (by keywords or by using a "seed" channel).
- Engagement and relevance analysis.
- AI-generated summaries of top results.
- Outreach email drafts in English or Spanish.

---

## 🚀 Features
- Search by **keywords** or by **seed channel**.
- Filter by subscribers, country, and recent activity.
- Calculate **engagement rates** and **keyword relevance**.
- Generate **AI summaries** of top creators.
- Draft **outreach emails** (English / Spanish).
